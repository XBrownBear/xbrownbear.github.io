使用 Maven 配置 SSM 框架

配置环境

  • 平台：

    Eclipse Mars.2、Windows 10、JDK 1.8、Tomcat 8.0、Maven 3.6、Navicat Premium 12（MySQL）

  • Eclipse配置：

    1. 配置编码格式 —– UTF-8；
    2. 配置JDK；
    3. 配置Maven；
    4. 配置Tomcat。

  • 项目框架配置：

    1. 新建Maven项目；

    2. 配置Project Facets：

      • Dynamic Web Module 2.5

        设置Runtimes –> Tomcat

      • java 1.8

      • javaScript 1.0

    3. build path配置：

      source –> 添加src/main/java（源代码）、src/main/resources（配置文件）、src/test/java（测试代码）

    4. 目录结构：

  • pom.xml配置：

    添加所需要的jar包。添加jar包的代码段参考 这里

  • 发布配置：

    项目属性 –> Deployment Assembly –> Add –> Maven…

配置数据库

配置web.xml

  1. 配置编码过滤器CharacterEncodingFilter；
  2. 配置MVC核心控制器dispatcherServlet；
  3. 配置ContextLoaderListener；
  4. 配置contextConfigLocation指定Spring配置文件路径。

配置spring-mvc.xml

  1. 启动自动扫描；
  2. 注册MVC注解驱动；
  3. 静态资源可访问的设置方式；
  4. 配置视图解析器。

配置db.properties

  • 连接数据库的参数

配置applicationContext.xml

  • 配置需要扫描的包；
  • 加载数据库参数文件并配置数据源；
  • 注册sqlSessionFactoryBean；
  • 配置Mapper扫描器。

配置log4j.properties

  • 配置日志文件

配置mybatis-config.xml

  • 设置日志以log4j进行显示;
  • 定义别名.

  • 实现Serializable 接口（序列化接口）；
  • getter/setter方法；
  • 构造方法（有参/无参）。
